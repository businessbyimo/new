# Facebook Personal Bot 🤖

আপনার ব্যক্তিগত Facebook আইডির জন্য একটি অটোমেটেড বট যা স্বয়ংক্রিয়ভাবে ফ্রেন্ড রিকোয়েস্ট গ্রহণ করে এবং মেসেজের রিপ্লাই দেয়।

## ✨ বৈশিষ্ট্যসমূহ

- ✅ স্বয়ংক্রিয় ফ্রেন্ড রিকোয়েস্ট গ্রহণ
- 👋 নতুন ফ্রেন্ড অ্যাড হওয়ার সাথে সাথে ওয়েলকাম মেসেজ
- 💬 নির্দিষ্ট কীওয়ার্ডের উপর ভিত্তি করে স্বয়ংক্রিয় রিপ্লাই
- 🔄 সংযোগ বিচ্ছিন্ন হলে অটো-রিকানেক্ট
- 🌐 হেলথ চেক এন্ডপয়েন্ট (Render-এর জন্য)

## 📋 প্রয়োজনীয়তা

- Node.js v18 বা তার উপরে
- একটি Facebook অ্যাকাউন্ট
- Render.com অ্যাকাউন্ট (ফ্রি)

## 🚀 দ্রুত শুরু

### ১. appstate.json তৈরি করুন
1. Chrome/Firefox-এ **C3C FbState** এক্সটেনশন ইনস্টল করুন
2. Facebook-এ লগইন থাকা অবস্থায় এক্সটেনশন চালু করুন
3. "Export State" বাটনে ক্লিক করুন
4. JSON কপি করে `appstate.json` ফাইল তৈরি করুন

### ২. লোকাল রান
```bash
# ডিপেন্ডেন্সি ইনস্টল
npm install

# বট চালু
npm start

# ডেভেলপমেন্ট মোড (nodemon)
npm run dev
